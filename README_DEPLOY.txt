Публикация на GitHub Pages

1. Создайте публичный репозиторий, например nancydesign24-site.
2. Загрузите в корень репозитория ВСЕ файлы из этой папки, не саму папку целиком.
3. В GitHub откройте Settings → Pages.
4. Source: Deploy from a branch. Branch: main, folder: / (root). Нажмите Save.
5. В поле Custom domain укажите nancydesign24.ru.
6. В Beget настройте DNS по значениям, которые покажет GitHub Pages.
7. После подтверждения DNS включите Enforce HTTPS.

Важно: index.html, assets, cases, styles.css и script.js должны лежать в корне репозитория.
